# Phong 光照模型

**注意**：视频加载需要时间

## 📋 项目简介
这是一个使用 Taichi 库实现的交互式 3D 渲染系统，展示了 Phong 光照模型的基本原理以及 Blinn-Phong 模型和硬阴影的效果。

## ✨ 功能特性

1. **Phong 光照模型**：实现了环境光、漫反射和镜面高光的计算
2. **Blinn-Phong 模型**：提供了 Blinn-Phong 光照模型的实现，使用半程向量计算高光
3. **硬阴影效果**：实现了基于阴影射线的硬阴影效果
4. **交互式参数调节**：通过 UI 控件实时调节材质参数
5. **光线投射渲染**：使用光线投射算法实现 3D 场景渲染
6. **深度测试**：实现了类似 Z-buffer 的深度竞争逻辑，保证正确的遮挡关系

## 📄 技术文档

### Phong 光照模型
Phong 光照模型将物体表面反射的光分为三个独立的计算分量：

1. **环境光**：模拟场景中经过多次反射后均匀分布的背景光
   $$I_{ambient} = K_a \times C_{light} \times C_{object}$$

2. **漫反射**：模拟粗糙表面向各个方向均匀散射的光，强度与光线入射角的余弦值成正比
   $$I_{diffuse} = K_d \times \max(0, \mathbf{N} \cdot \mathbf{L}) \times C_{light} \times C_{object}$$

3. **镜面高光**：模拟光滑表面反射的强光，强度与观察方向和理想反射方向的夹角有关
   $$I_{specular} = K_s \times \max(0, \mathbf{R} \cdot \mathbf{V})^n \times C_{light}$$

### Blinn-Phong 模型
Blinn-Phong 模型是对 Phong 模型的改进，使用半程向量 \(\mathbf{H}\) 代替反射向量 \(\mathbf{R}\)：

$$\mathbf{H} = \frac{\mathbf{L} + \mathbf{V}}{|\mathbf{L} + \mathbf{V}|}$$

$$I_{specular} = K_s \times \max(0, \mathbf{N} \cdot \mathbf{H})^n \times C_{light}$$

**视觉差异**：Blinn-Phong 模型在高光区域边缘更加柔和，尤其是在大入射角时，高光效果更加集中和自然。

### 硬阴影实现
硬阴影通过以下步骤实现：
1. 从每个交点向光源方向发射一条阴影射线
2. 检查阴影射线是否在到达光源前击中其他物体
3. 如果击中，则该点处于阴影中，只计算环境光

### 光线投射算法
1. 为屏幕上的每一个像素发射一条射线
2. 计算射线与场景中几何体的交点
3. 选择距离摄像机最近的交点进行着色
4. 在交点处计算法向量和光照效果

### 技术发现

1. **Blinn-Phong 与 Phong 模型差异**：
   - Blinn-Phong 模型在高光区域边缘更加柔和，尤其是在大入射角时
   - 高光效果更加集中和自然，减少了传统 Phong 模型在某些角度下的失真

2. **硬阴影实现**：
   - 阴影射线的发射和检测是实现硬阴影的关键
   - 阴影中的点只计算环境光，使阴影区域明显暗于其他区域
   - 光源位置和几何体布局对阴影效果影响显著

## 注意事项
- **向量归一化**：参与点乘的 \(\mathbf{N}\)、\(\mathbf{L}\)、\(\mathbf{V}\) 必须是单位向量
- **负值处理**：使用 `ti.max(0.0, dot_product)` 截断负值，避免非法运算
- **颜色限制**：使用 `ti.math.clamp(color, 0.0, 1.0)` 将颜色强制限制在合法区间内
